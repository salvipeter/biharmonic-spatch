Practical S-patches

Introduction [0.5]:
- az S-patch egy matematikailag gyonyoru megoldas egy fontos problemara, blabla...
- ... de a rengeteg kontrollponttal kezelhetetlen
- lehet valamilyen energiat minimalizalni, de mivel egy-egy pont hatasa kicsi, a kontrollpont eloszlas nem lesz szep - be lehet vezetni valami controlnet fairing tagot, de azert az nem az igazi

Previous work [0.5]:
- Farin-fele diszkret Coons-patch otlet: egy lokalis maszkot alkalmazunk a kontrollhalora, amitol a felulet (is) harmonikus / minimalis lesz (de ld. Monterde, aki megmutatja, hogy ennyire nem egyszeru a helyzet)
- Kosinka-fele beallitasa a szelso kontrollpontoknak a G1-hez + belseje
  => csak cubic
     ok is valami konvex kombinaciot hasznalnak a belso pontokra, de nincs reszletezve
- a 90-es S-patch cikkben leirt G1-beallitas es default belso beallitas
  => bonyolult, es csak masod- es harmadfoku van jol leirva (kulonosen belso pontok)
- transzfinit es GB patchekkel valo osszehasonlitas
  => transzfinittel szemben tobb DoF, de a GB patchnel bonyolultabb
  => a GB patchnel magasabb fokszam/oldalszam eseten a sulyok nem a legjobbak,
     rosszabb approximaciora?

Preliminaries [1]:
- az S-patch leirasa
- indexeles
- melyek a szelen levo kontrollpontok
  => minden szelen levo nem sarok kontrollpont ket belso ponthoz csatlakozik,
     ezt a ket pontot egybehuzzuk (=> normalvektorsopresrol tudunk vmit mondani?)
- mikor lesz ket kontrollpont szomszedos

Generalized Masks [1]:
- 4-oldalunal a G1 maszk bemutatasa (ld. Monterde):
  0 0 c 0 0
  0 b a b 0
  c a O a c    ahol a=8/20, b=-2/20, c=-1/20, es O=-1
  0 b a b 0
  0 0 c 0 0
- ez a maszk ugy jon letre, hogy vesszuk az eredeti harmonikus maszkot:
  0 a 0
  a O a        ahol a=1/4, es O=-1
  0 a 0
  .. es alkalmazzuk onmagara, majd normaljuk a kozepso elemre
- ugyanezt felirjuk S-patch jelolessel - az 'a' pontok megfelelnek a szomszedoknak, a tobbi az ismetelt alkalmazasbol kiadodik
- a bonyolult pontok kozti kapcsolat miatt nem ugyanolyan maszkot kapunk egy adott S-patchen belul sem (pl. mar egy belso pont kozvetlen szomszedainak a szama sem fix, hanem az indexben szereplo nem 0 elemek szamanak a 2x-ese)
